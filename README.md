## q-chrome

Promise-based [chrome api](https://developer.chrome.com/extensions/api_index#stable_apis).

### install

```bash
npm install q-chrome
```

or

```bash
git clone https://github.com/acvetkov/q-chrome.git
cd q-chrome
npm install
```

### build

```bash
npm run build
```

### test

```bash
npm run test
```
